# 项目：

以Zilo Whisper作为输入设备, 以Viaaim AI耳机和IMessage/Telegram等作为输出设备的智能个人Agent助手，基于区块链技术保证用户数据存档和隐私安全不依赖中心服务器（或中心服务器仅作为AI转发，数据存储不由中心存储）并提供Connector用于接入BiliBili、Apple Health、Live Translate等技术

1. 用户通过戒指唤醒 Agent；viaim 耳机接收语音；Agent 在耳边追问；用户继续用语音回答；Agent 执行并通过耳机确认。

2. 用户不下载新 App；直接给 Agent 发消息；Agent 可以主动跟进；用户能够在原消息里继续修改任务；Agent 能进入群聊；在多个消息渠道保持同一个身份和上下文。

# MVP：

## 1\.PAWN：面向 B站创作者的无屏伴身 Agent

用户在走路、拍摄或活动现场突然产生灵感时：

用 **Zilo 戒指按钮**唤醒 PAWN； 

通过戒指麦克风或 viaim 耳机说出灵感； 

PAWN 在 **viaim 耳机中进行至少三轮追问**； 

用户通过戒指手势确认、取消或标记隐私级别； 

PAWN 生成适合 B站的视频大纲、标题、分镜和提词稿； 

加密内容保存在本地，内容哈希和授权状态写入 **Injective 测试网**； 

生成结果通过 **Photon 发送到真实 iMessage**； 

用户直接在 iMessage 中回复修改，PAWN 保持同一任务上下文； 

拍摄时通过戒指手势控制耳机提词的下一句、重复和暂停。 

一句话：

> **从指尖捕捉灵感，在耳边完成创作，通过消息持续协作，并让每一段记忆始终归用户所有。**
